Employee Appraisals and Evaluation

Employee Evaluation Process - Odoo App

This app allows your HR / Evaluation Team to do Employee Evaluation Process.


Main Features:

  • Configuration of Appraisals/Evaluation Plans

  • Evaluation users can create Appraisals/Evaluation for employees.

  • Interview Requests will be sent to Employee, Employee Manager and Employee Downlines.

  • Top-Down, Bottom-Up, Self and Final Appraisal Plans can be setup.

    • Top-Down :- Employee Manager can do the Appraisal Interview Request.

    • Bottom-Up :- Employee Downline can do the Appraisal Interview Request.

    • Self :- Employee can do the Appraisal Interview Request.

    • Final :- Employee Manager can do the Final Evaluation Appraisal Interview Request.

  • Interview Requests will be created for every level of employee during evaulation process based on plan of employee appraisal.

  • During the interview request interviewer can fill out a survey form (Survey is from Odoo standard / out of box feature) and submit a survey. At the end evaluation manager or user can also print survey which is also out of the box feature of Odoo survey module.

  • Next appraisal date on employee form and also allow you to set Appraisal plan on Employee form.

  • For more details please watch video and screenshots.

  • This app is only for Odoo Community Edition. So please make sure before buying that you are using Odoo community edition.
